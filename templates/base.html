API Documentation
Endpoints

The API provides the following endpoints:

POST /api/optimize

Optimize fund allocation based on provided Excel data and parameters.

Parameters (multipart/form-data):
  • file - Excel file with bank data (required)
  • total_funds - Total funds available (default: 1,000,000)
  • investment_duration - "short", "mid", or "long" (default: "mid")
  • cdars_interest - "true" or "false" (default: "false")
  • multi_terms - "true" or "false" (default: "false")
  • single_bank_mode - "true" or "false" (default: "false")
  • single_bank - Bank name when single_bank_mode is true
Response:
{
  "success": true,
  "message": "Optimization completed successfully",
  "results": [ ... array of allocation results ... ],
  "summary": {
    "total_allocated": 1000000,
    "total_return": 25000,
    "weighted_avg_rate": 2.5
  },
  "bank_count": 3,
  "term_count": 4
}
POST /api/banks

Get a list of available banks from an Excel file.

Parameters (multipart/form-data):
  • file - Excel file with bank data (required)
Response:
{
  "success": true,
  "banks": ["Bank A", "Bank B", "Bank C"]
}
